Detailed Description

ROS node for setting/cancelling robot targets.

Initializes a ROS node responsible for setting and managing robot targets. It communicates with the Action Server '/reaching_goal' using an action client, subscribes to '/robot_target' , '/odom' ,and '/robot_cancel_goal' topics, and publishes the '/robot_state'. Additionally, it sets up a timer to periodically update the robot's actual pose.
